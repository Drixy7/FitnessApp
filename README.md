### Installation

1. **Clone repository**

2. **Fetch dependencies:**
   ```bash
   flutter pub get
   ```
3. **Generate Isar schemas (if required):**
    ```bash
    flutter pub run build_runner build
   ```
4. **Run the app:**
    ```bash
    flutter run
   ```
